# Remote Workspace Layout

This repository keeps a public-safe skeleton of the remote workspace used for the StereoPilot-lite reproduction. Large artifacts are represented by README placeholders in their expected locations.

## Project Directories

| Path | Included in Git | Runtime role |
| --- | --- | --- |
| `StereoPilot/` | Code, configs, small assets, sample inputs, placeholders | Official inference-side snapshot and checkpoint location. |
| `diffusion-pipe-stereo/` | Code, configs, training tools | diffusion-pipe-derived StereoPilot-lite training code. |
| `models/` | Placeholder only | Optional local symlink/pointer area for shared model paths. |
| `data/` | Placeholder skeleton only | Prepared left/right training clips. |
| `datasets/` | Placeholder skeleton only | Raw downloaded/extracted Stereo4D-derived source data. |
| `runs/` | Placeholder skeleton only | Training logs, QA reports, DeepSpeed checkpoints, LoRA adapters. |
| `stereopilot_exports/` | Placeholder skeleton only | Merged safetensors, inference test TOMLs, generated outputs. |

## Not Part Of The Project Snapshot

The remote machine also contained system/runtime directories such as CUDA installers, Conda environments, shell caches, editor servers, and OS-managed folders. Those are intentionally excluded because they are machine setup state, not project source.

`wan_stereo_remote_bundle/` was a historical bootstrap bundle used while installing the first remote copy. Its relevant code is now integrated into `diffusion-pipe-stereo/`, `docs/`, and `scripts/`, so it is not required for a fresh clone.

## External Artifacts To Restore After Clone

| Artifact | Place it here | Why |
| --- | --- | --- |
| Wan2.1 T2V 1.3B checkpoint directory | `StereoPilot/ckpt/Wan2.1-T2V-1.3B/` or update TOML paths | Required for training and LoRA export. |
| Official `StereoPilot.safetensors` | `StereoPilot/ckpt/StereoPilot.safetensors` | Optional for official inference/reference shape checks. |
| Prepared stereo clips | `data/<dataset>/left` and `data/<dataset>/right` | Required for training. |
| Raw Stereo4D-derived HF tar/extracted videos | `datasets/stereo4d_hf_raw/` | Optional if regenerating prepared clips. |
| LoRA adapters and DeepSpeed checkpoints | `runs/` | Generated by training. |
| Merged StereoPilot-compatible weights | `stereopilot_exports/` | Generated by the exporter. |
